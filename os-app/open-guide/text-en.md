# Quick Introduction

Transformations are specified as an *Expression*.

An *Expression* begins with a *Variable*, usually `$input`.

An *Operation* follows the *Variable*, for example `lines`:

```
$input.lines
```

Sometimes an *Operation* takes arguments

```
$input.prepend(Hello )
```
